# Support

## 如何获取帮助

如果你在使用 Meliora 时遇到问题，可以通过以下方式获取帮助：

### 1. 查看文档

- [README.md](README.md)：项目介绍和基本使用说明
- [agent.md](agent.md)：代码风格和设计规范
- [CONTRIBUTING.md](CONTRIBUTING.md)：贡献指南

### 2. 搜索 Issues

在 [GitHub Issues](https://github.com/abloom25/Meliora/issues) 中搜索类似的问题，可能已经有解决方案。

### 3. 创建 Issue

如果没有找到解决方案，可以创建新的 Issue：

- **Bug 报告**：使用 [Bug Report 模板](.github/ISSUE_TEMPLATE/bug_report.md)
- **功能请求**：使用 [Feature Request 模板](.github/ISSUE_TEMPLATE/feature_request.md)

### 4. 社区讨论

通过 GitHub Discussions 与其他用户交流。

## 常见问题

### Q: 如何配置远程歌单？

### Q: 如何配置本地音乐？

请参考 [README.md](README.md) 中的"GitHub Pages 自动部署"部分。

### Q: 如何部署到 Vercel？

请参考 [README.md](README.md) 中的"Vercel 自动部署"部分。

### Q: 遇到构建错误怎么办？

确保你已经安装了所有依赖，并且运行了正确的命令：

```powershell
pnpm install
pnpm build
```

如果问题仍然存在，请创建 Issue 描述错误信息。
